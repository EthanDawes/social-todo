# Social TODO

A social media spoof (not affiliated or endorsed by any company) to make doomscrolling more productive: remind you of your tasks.

## Apps
- Tumblr
- 4chan
- Twitter

## Usage
Click one of these app links, then "Add to homescreen" from your browser

## Developing

### Frontend
```bash
cd frontend
pnpm install
pnpm dev
```

### Backend
```bash
pdm install
cd backend
```
